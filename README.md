# CYW OS — Control Your World

**Sovereign AI Orchestration Architecture**

CYW OS is the technical architecture site for the Control Your World orchestration layer. It describes the system design at a marketing-plus-light-technical level of abstraction.

**U.S. Provisional Patent Application No. 63/980,310 — Patent Pending**

## Stack

- [Next.js](https://nextjs.org/) 14
- [Tailwind CSS](https://tailwindcss.com/) 3.4
- [TypeScript](https://www.typescriptlang.org/) 5.4
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Connected to Vercel. Push to `main` to deploy.

- **Production domain:** cyw-os.com
- **Framework:** Next.js (auto-detected by Vercel)
